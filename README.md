# Quarter 3 - Python Learning

## Overview
In this quarter, I am focusing on learning Python. This quarter is dedicated to understanding Python's use in AI and software development.

## Learning Topics
I am working on the following topics this quarter:

1. **Python Basics**
   - Syntax and Variables
   - Data Types (Lists, Tuples, Dictionaries, Sets)
   - Control Structures (if-else, loops)

2. **Object-Oriented Programming (OOP)**
   - Classes and Objects
   - Inheritance and Polymorphism
   - Encapsulation and Abstraction

3. **Data Handling & File Management**
   - CSV, JSON file handling
   - Using Pandas library
   - Database connectivity (SQLite, PostgreSQL)

4. **Python for AI & Machine Learning**
   - Using NumPy and Pandas
   - Data visualization with Matplotlib and Seaborn
   - Introduction to Scikit-learn and TensorFlow

5. **API Development & Web Scraping**
   - Using Flask and FastAPI
   - Web scraping with BeautifulSoup and Scrapy

6. **Automation & Scripting**
   - Bash scripting
   - Using Selenium and PyAutoGUI
